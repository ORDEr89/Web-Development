CSS Box Model & Display Property

1. CSS Box Model Demonstration

This is the content area
Box Model Breakdown (for the yellow box above):
• Content → 300px × 140px
• Padding → 25px on all sides (adds 50px to width & height)
• Border → 8px on all sides (adds 16px to width & height)
• Margin → 40px (space outside the border)

Total space occupied = margin + border + padding + content
→ Width: 40 + 16 + 50 + 300 = 406px
→ Height: 40 + 16 + 50 + 140 = 246px
